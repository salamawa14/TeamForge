# TeamForge
SE302 Software Project Management Course Project  A web-based Project Matching and Team Formation Platform that helps university students create projects, find teammates based on skills, and connect with academic advisors.

##  User Roles
[cite_start]The system supports three distinct user roles, each with specific functionalities[cite: 35, 36]:
* [cite_start]**Admin:** Manages project categories, user roles, and publishes official announcements regarding deadlines[cite: 69, 74, 81].
* [cite_start]**Instructor (Advisor):** Manages a professional profile, receives and evaluates supervision requests from students[cite: 352, 355].
* [cite_start]**Student:** Creates project announcements, searches for teammates, and applies to existing projects or supervision[cite: 436, 446, 508].

##  Key Features
* [cite_start]**Project Matching:** Skill-based filtering to help students find the right partners for their projects[cite: 33, 53].
* [cite_start]**Request System:** A dedicated flow for sending, accepting, or rejecting membership and supervision requests [cite: 60, 65, 445-446].
* [cite_start]**Profile Management:** Detailed profiles for students and instructors highlighting technical skills, interests, and availability[cite: 88, 202, 361].
* [cite_start]**Announcements:** A centralized hub for admins to post important dates and requirements[cite: 162, 177].
